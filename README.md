# Hausarbeit-GenAI

For reproduction of the code please consider the following notes: 

## Run code Locally
Execute code in the given order. The first code files (.py) have been executed on a local machine and contain exclusively data pre-processing and inspection of the data to constitute the final dataset. The raw data from LexisNexis (~5GB) can not be provided due to size restrictions of github. If needed contact me via Email. 

- df_newspaper_filtered_by_paragraph_mergedAB.csv (full dataset)
- gold_annotated.csv (subsample of human annotated data)

Before executing code, set your working directory individually.

## Run code in Google Colab
Execute following code (.ipynb) in Google Colab. Upload the data into Google Drive home directory (/MyDrive). 

The gold_annotation_script.py is a script for easier annotating newspaper paragraphs and does not need to be reproduced.
